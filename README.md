# Exam Grading Application

Это WPF-приложение предназначено для расчета суммы баллов за три модуля экзамена и определения оценки по 5-балльной шкале. Приложение также включает модульные тесты для проверки бизнес-логики.

## Описание функционала

1. **Ввод данных**:
   - Пользователь вводит баллы за три модуля:
     - **Модуль 1**: Разработка, администрирование и защита баз данных (максимум 22 балла).
     - **Модуль 2**: Разработка модулей программного обеспечения (максимум 38 баллов).
     - **Модуль 3**: Сопровождение и обслуживание программного обеспечения (максимум 20 баллов).

2. **Обработка данных**:
   - Приложение проверяет корректность введенных данных.
   - Суммирует баллы и определяет оценку по следующей шкале:
     - **56–80 баллов**: "5 (отлично)".
     - **32–55 баллов**: "4 (хорошо)".
     - **16–31 баллов**: "3 (удовлетворительно)".
     - **0–15 баллов**: "2 (неудовлетворительно)".

3. **Вывод результата**:
   - Результат выводится в текстовом блоке в формате:  
     `Результат: [сумма баллов] баллов, оценка: [оценка]`.
